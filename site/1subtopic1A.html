This website built using a costum static site generator i forked from [tendigits] and modified for my own purposes. The website is hosted on github. To add new entries i duplicate existing folder in src/content directory, edit the content.htm file, save and run the build.sh script. 


    #!/bin/sh -e
    site=../site
    content=content
    headerA=../../inc/header-top.htm
    headerB=../../inc/header-bottom.htm
    sitenav=../../inc/nav.htm
    meta=meta.htm
    contentFile=content.htm
    foot=../../inc/footer.htm
    bottom=../../inc/html-bottom.htm
    tally=0

    
    rm -rf $site
    mkdir -p $site
    
    # List the index
    function setupindex {
        echo "<h1>Full site index</h1>" > index/content.htm;
        for f in *; do
            if [ $f != 'index' ]; then
                echo "<a href='${f}.html'>${f}</a><br>" >> index/content.htm;
            fi
        done
        echo "Index built"
    }
    
    # Setup topics
    cd $content
    setupindex;
    for f in *; do
        cd $f;
        markup=''
        topPart=$(cat $headerA $meta $headerB);
        nav=$(cat $sitenav);
        contentText=$(cat $contentFile);
        footer=$(cat $foot);
        closefile=$(cat $bottom);
        mainContent="<main>${contentText}</main>";
        sideBar="<aside>${markup}</aside>";
        echo ${topPart}${nav}"${mainContent}"${sideBar}${footer}${closefile} > ../../$site/${f}.html
        cd ..
        tally=$((tally+1))
    done
    echo "${tally} topics built"

It combines the .htm from /content and /inc dir. The output is properly formatted .html files in /site.

Fyi to display html code inside html you just need to replace all < with & lt;(& and lt; writen together), Still looks the same but the code wont execute.
